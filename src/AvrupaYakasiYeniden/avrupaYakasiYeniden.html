Avrupa Yakasi Dergi Ofisi v2

The reason I made "Avrupa Yakasi Dergi Ofisi v2" is because Diorama artist Ahmet Gezer and I decided to create a physical copy of this work. Therefore, I reorganized the project and remodeled it from top to bottom.

The models are much more stylized in this version.

I adopted a subdiv workflow instead of a bevel workflow. This is because it doesn't cause many problems when I want to rescale this project and it contributes to its stylized appearance.
